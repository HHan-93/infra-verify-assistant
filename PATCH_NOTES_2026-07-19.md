## 🆕 신규 기능

- **실시간 로그 tail 뷰어**: 툴바에서 바로 열어 원격 서버의 로그 파일을 `tail -f`처럼 실시간으로 확인. 경로를 몰라도 되도록 자동완성 입력, 트리 구조 "찾아보기"(기본 `/var/log`), 자주 쓰는 로그 프리셋 칩(커스텀 추가/삭제 가능), 최근 사용 경로(삭제 가능) 제공. root 권한 필요 시 sudo 비밀번호 입력 지원.
- **AI 자연어 명령어 생성**: AI 패널에 [일반 대화]/[명령어 생성] 모드 탭 추가. "디스크 공간 확인해줘" 같은 자연어 요청을 셸 명령어로 변환해 카드 형태(명령어 + 설명 + 입력/실행 버튼)로 제공.
- **원격 ↔ 원격 세션 간 직접 파일 전송**: 파일탐색기 좌측 패널을 로컬 대신 다른 연결된 세션으로 전환해, 두 원격 서버 사이에 파일/폴더를 바로 전송(내부적으로 임시 폴더 경유 중계).
- **전체 프로세스 뷰**: 대시보드에서 상위 프로세스 외에 전체 프로세스를 정렬·검색하며 확인하고 필요 시 종료(kill).
- **SFTP 원격 디렉토리 재귀 파일명 검색**: 파일탐색기 원격 패널에서 하위 폴더까지 포함한 파일명 검색.
- **임의 분할 레이아웃(tmux 스타일)**: 좌우/상하/4분할 원클릭 버튼을 기본으로 제공하고, 필요 시 현재 패널을 임의로 추가 분할(가변 개수/비율)하는 보조 옵션 제공.
- **SFTP 듀얼패인 + 다중선택**: 로컬↔원격 파일을 나란히 보며 다중 선택 기반 배치 업로드/다운로드/삭제.
- **다중 실행 결과 검색/필터**: 여러 세션에 명령 1회 실행 후 모은 결과표에서 키워드로 검색/필터.
- **세션 로그 뷰어**: 세션별 로그 기록을 목록/검색하고 터미널 화면 그대로 리플레이. 보관 기간(일)·최대 개수를 뷰어 안에서 직접 설정 가능.
- **사용자 정의 프리셋 명령어 / 시나리오**: 추가·편집·삭제, 순서 변경(드래그앤드롭, 자유 위치 이동), 패널 좌우/상하 리사이즈.
- **프로필 가져오기/내보내기**: CSV/JSON 형식으로 접속 프로필을 내보내고 다시 가져오기(예시 가이드 행 포함).
- **모니터링 이력 장기 보관 / AI 대화 히스토리 유지**: 앱 재시작 후에도 과거 모니터링 데이터와 AI 대화가 이어짐.
- **마스코트(픽셀 판다) 애니메이션 리뉴얼**: 유휴 상태에서 대나무 먹기/노트북 타이핑/안경 고쳐쓰기 등 일상 소품 동작이 랜덤 루프에 섞여 재생. 터미널 클릭 시 놀람, 세션 접속 끊김 시 슬픔 리액션(마스코트가 이미 나와있을 때만 동작). 전 구간에 Squash & Stretch 적용으로 움직임이 더 자연스러움.

## ⚙️ 기능 고도화 / UX 개선

- 그리드(다중 세션) 모드에서 세션 클릭 시 연결 폼이 열렸다 닫혔다 깜빡이던 문제를 개선 — 셀별 인라인 연결 폼으로 분리.
- 그리드 세션 개수에 맞춰 늘어났던 빈 세션 탭이, 단일 보기로 전환 시 자동으로 정리됨(연결된 세션은 유지).
- 실시간 로그 tail 뷰어 아이콘을 와이파이처럼 보이던 아이콘에서 다른 아이콘으로 교체하고, 툴바 내 위치를 로깅/세션 로그 뷰어 버튼 사이로 이동.
- AI 패널의 "선택 AI 분석"/"전체 세션 AI 분석" 실행 시, 결과 형식(일반 대화)과 탭 표시가 어긋나 보이던 부분을 탭 자동 전환으로 정합성 개선.

## 🐛 버그 수정

- CSV/JSON 내보내기 및 템플릿 파일에서 한글이 깨져 보이던 문제(UTF-8 BOM 누락) 수정.
- 세션 로그 리플레이 시 터미널이 창 크기에 맞춰지지 않던 문제 수정.
- 임의 분할 레이아웃에서 탭을 닫을 때 분할 트리가 정리되지 않아 빈 화면(패널)이 남던 문제 수정.
- 실시간 로그 tail 뷰어에서, 같은 세션 안에서 "다른 경로"로 재시작할 때 이전 스트림의 늦게 도착한 데이터/종료 이벤트가 새 화면에 섞여 들어가던 문제 수정.
- AI 패널의 "선택/전체 세션 AI 분석" 버튼에 스트리밍 중 재요청 방지 처리가 빠져 있어, 연속 클릭 시 이전 응답이 멈춰버리던 문제 수정.
- 파일탐색기에서 원격→원격 전송 시, 세션 전환 후에도 이전 세션 선택이 남아있거나 자기 자신에게 전송을 시도할 수 있던 문제 수정.
